README - Basit Anket Yönetim Paneli (PHP + MySQL)

KURULUM (Hoca için tek seferlik 30 sn)
1) XAMPP kur -> Apache + MySQL START
2) phpMyAdmin aç: http://localhost/phpmyadmin
3) Import -> survey_app.sql dosyasını içe aktar
4) Proje klasörünü şuraya koy:
   C:\xampp\htdocs\survey-panel
5) Aç:
   http://localhost/survey-panel/
